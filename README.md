# SURO Keyboards

Premium keyboard landing page built around soft motion, product-focused imagery, and a calm blue visual system.

## Overview

SURO Keyboards is a Vite + React showcase for a mechanical keyboard brand. The page includes a full-screen hero, animated headline, product feature strip, image-led detail gallery, product carousel, and cookie preference banner.

## Features

- Full-screen animated hero with typewriter headline
- Product keyboard composition with layered brand visuals
- Feature strip for acoustics, switches, battery, devices, and hot-swap PCB
- Three-panel product detail gallery
- Infinite-style product carousel with side previews
- Cookie consent banner with accept, decline, and preferences actions
- Responsive layout for desktop and mobile

## Tech Stack

- React 19
- Vite 6
- Lucide React icons
- Custom local fonts and image assets

## Getting Started

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
